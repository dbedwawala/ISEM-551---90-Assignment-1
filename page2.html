Multiple Input Multiple Output (MIMO)

Overview of MIMO technology

i) Uses multiple antennas to transmit and receive radio signals in order to improve the overall radio performance.

ii) The most widely used antenna configuration with four transmit and four receiving antennas.

iii) The use of MIMO overcomes the problems of multi path fading and interference.

iv) Other types of antenna technology are single input single output (SISO), single input multiple output (SIMO) and multiple input single output (MISO). However, LTE makes use of MIMO for the efficient radio transmission.
