Brief description on 4G technology

i) 3GPP defined 4G technology also known as Long Term Evolution (LTE) to provide higher data rate and lower latency to end user.

ii) The fourth generation technology uses Multiple Input Multiple Output (MIMO) to provide higher throughput to the end user by using the multiple beams from the antenna.

iii) LTE uses Orthogonal Frequency Division Multiplexing (OFDM) to achieve higher spectral efficiency.

iv) LTE provides a scalable bandwidth ranging from 1.5 to 20 MHz so that the carrier has an option to make a cost effective network while deploying LTE.

v) LTE is a fully packet switched network and it can provide a theoretical throughput of 300 Mb/s on downlink and 75 Mb/s on the uplink.
